Instructions:

  1. Select libraries
  2. Enter project name
  3. Click "Generate"
  4. Paste command into comand line

Notes:

  • Output: The script creates a new folder <project_name> in your current working directory. The resulting folder structure is as follows: 
    - <project_name>
  - .gitignore
  - README.md
  - CMakeLists.txt
  - src
    - main.cc
  - extern
    - your_submodule_1
    - your_submodule_2
  - bin
  • Existing Repository: Some people prefer to first create a remote git repository before initially committing to it. If that is the case, you can enter its git url in the "Existing Repository (optional)" field. The repository will then be cloned first and the project will be configured inside.
  • SSH: Default method for cloning submodules is https. Check the SSH checkbox to use your ssh key instead.
